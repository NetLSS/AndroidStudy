# IPC

- 액티비티에서 실행중인 서비스를 제어하거나 데이터를 사용하는 등의 작업이 필요할 때
- 현재 실행중인 서비스에 접속하고 서비스가 가지고 있는 메서드를 호출할 수 있는 개념
- 서비스가 돌고있고, 액티비트를 실행시켜서, 서비스 메서드를 사용하고, 액티비티가 그걸 받아서 사용. (이후 액티비티 꺼도 서비스는 돌고 있는 것)

- 현재 실행되고 있는 서비스에 접속할 수 있는 개념
